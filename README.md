# 🔢 Number Systems Converter

A Python command-line tool that converts numbers 
between Decimal, Binary, Hexadecimal and Octal.

## 🔧 Features
- Decimal to Binary, Hex and Octal
- Binary to Decimal
- Hexadecimal to Decimal
- Octal to Decimal
- Full conversion table for every result
- Input validation and error handling

## 🚀 How to Run
```bash
python converter.py
